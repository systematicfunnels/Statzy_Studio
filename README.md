## 🎥 What is Statzy Studio?

**Statzy Studio** is a personal AI framework I built to fully automate video creation — from script to final render.

It handles everything:

* Idea → Script
* Script → Voiceover
* Voiceover → Visuals
* Visuals → Final video

All with minimal manual effort.

---

## 🔥 What It Can Do

* 🎬 Generate complete videos automatically
* 🗣️ Create realistic voiceovers in **30+ languages**
* 🌍 Dub videos into different languages
* ✂️ Edit and assemble clips intelligently
* 🧠 Maintain memory across workflows
* 📱 Produce short-form content (Reels, Shorts, TikTok)

---

## ⚙️ How It Works

Statzy Studio is built as a modular pipeline:

```text
Idea → Script → Voice → Assets → Editing → Final Video
```

Each step is powered by AI and can be customized or automated further.

---

## 🧠 Core System

* **Short Video Engine** → creates viral-style short content
* **Long Video Engine** → builds structured full-length videos
* **Translation Engine** → converts videos into other languages
* **Editing Engine** → runs a programmable editing workflow

---

## 🛠️ Tech Behind It

* MoviePy (video processing)
* OpenAI (LLM workflows)
* ElevenLabs / EdgeTTS (voice synthesis)
* Pexels API + Bing (media sourcing)
* TinyDB (lightweight memory system)

---

## 🚀 Why I Built This

I wanted a system that removes the repetitive parts of content creation —
so I can focus on ideas instead of editing.

Statzy Studio is that system.
